# articles

- convert jupyter notebook to md file

    *jupyter nbconvert --to markdown \<name of jupyter file\>*

- upload the article to medium.com

    1. create document in jupyter notebook
    2. convert in to MD
    3. preview MD document
    4. copy and paste into medium.com web app
    5. check if medium.com didn't split code fields, if yes merge them into one using keyboard delete to remove lines. Fields will be automatically joined.

- tips for deploying to medium.com
    1. for plain code use markdown jupyter notebook cell with *pre* html tag
